## Core Principles

### 1. Tapes Are Fractal Systems
Every tape is self-contained and can:
- Run standalone
- Be loaded by another tape
- Load other tapes
- Broadcast a REST API

### 2. GHOST Enables Communication
The Global Host Orchestration Specification for Tapes (GHOST) defines:
- How tapes register
- How tapes communicate
- How agents route
- How external services integrate

### 3. HOLO UI Is Universal
All tapes share the green phosphor CRT aesthetic:
- Consistent visual language
- Scanlines and glow effects
- Monospace typography
- Card-based layouts
